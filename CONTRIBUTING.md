# Contributing

Thanks for your interest in Osinaldi BluRay MultiBurner.

For bug reports, please include:

- Linux distribution and version
- Desktop environment
- Python/Tk version if known
- Number and model of optical writers
- Terminal output or logs from `~/OsinaldiBurnLogs/`
- Steps to reproduce the issue
